Method fire ¶

This method gets event object and does call URI event handlers. 

JSNAV.fire(<Event> event) : undefined


Features: ¶

  • call event handlers for event.event
  • if event.anchor is defined and anchor element is found then method makes scrollTo to anchor element
  • if event.event is defined then method calls all registered handlers for event named event.event
  • handlers are executed in context of window.document.body or anchor element if it is found
  • handlers are executed sequentially, in order of their registration for this event by bind
