Method unbind ¶

This method removes event handlers. 

JSNAV.unbind(<string> navEventName, <Handler)> handler = undefined) : undefined


Features: ¶

  • JSNAV.unbind(navEventName)

will remove all handlers for an event with that navEventName

  • JSNAV.unbind(navEventName, handler)

handler will be unbind for event with navEventName name
