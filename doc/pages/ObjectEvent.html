Object Event ¶

this object Event is uses in JSNAV 

{
  anchor: anchorName,
  event: eventName,
  params: [array of unescaped params]
  }
  • anchor - 'undefined' or anchor name
  • event - 'undefined' or name of the event (name is decoded from the URL encoding; only UTF-8 supports in encoding)
  • params - always array; consist of parameters from URI
