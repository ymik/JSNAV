Handler function ¶

Handler is javascript function with some string parameters. 

function(<string> param1, <string> param2, ...){
   ...
}

This function is calls by JSNAV in context of AnchorElement or window.document.body

Example: ¶

var addClassHandler = function(className, textId) {
    var elm = textId? document.getElementById(textId) : this;
    if (elm.className && elm.className.indexOf(className) < 0) {
        elm.className += " " + className;
    }
}
JSNAV.bind("addClass", addClassHandler);
