JSNAV form of URI: ¶

http://user:pass@host:port/path/resource?query#anchor;navEventName;param1;param2;...


Syntax of the anchor: ¶

  1. before special symbol # URI is no different from that specified in RFC1630, the changes relate only to the syntax anchor
  2. immediately after the # symbol and up to symbol ; place for a classical HTML anchor with the standard action (scroll the page to the anchor element)
  3. after first symbol ; and before next ; (or to end of anchor) is place for event name
  4. and ;-separated parameters list at end of URI
