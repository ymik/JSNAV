Method bind ¶

This metod binds handler to URI event. 

JSNAV.bind(<string> navEventName, <Handler> handler) : undefined


Features: ¶

  • for one event name, you can add any number of handlers; they will be called sequentially, in order of registration
  • for removing handlers call unbind


Example ¶

adding handler: 

var addClassHandler = function(className, textId) {
  var elm = textId? document.getElementById(textId) : this;
  if (elm.className && elm.className.indexOf(className) < 0) {
  elm.className += " " + className;
  }
  }
  JSNAV.bind("addClass", addClassHandler);

This addClassHandler handler will be invoked on the next URIs:

  • test.html#;addClass

calls addClassHandler() in context of window.document.body

  • ttest.html#;addClass;className

calls addClassHandler('className') in context of window.document.body

  • ttest.html#;addClass;className;elementId

calls addClassHandler('className', 'elementId') in context of window.document.body

  • ttest.html#anchor;addClass;className

calls addClassHandler('className') in context of the AnchorElement and scroll to AnchorElement
